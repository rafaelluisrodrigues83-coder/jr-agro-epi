# JR Agro EPI — Sistema de Gestão

Demo interativa do sistema de gestão de EPI agrícola.

## Senha de acesso
`jragro2025`

## Como fazer o deploy no Netlify

1. Faça upload desta pasta no GitHub
2. Conecte o repositório no Netlify
3. O Netlify detecta automaticamente as configurações
4. Acesse o link gerado e use a senha acima

## Estrutura
- `src/App.js` — controle de login
- `src/Login.js` — tela de senha
- `src/Sistema.jsx` — sistema completo
- `netlify.toml` — configuração de deploy
